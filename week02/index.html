Scope: Var vs Let & Const

Var was used in the past but did not have a block scope. Using let and const protects you from overwriting any built-in methods by assignment.

Vocabulary & Functions

Array Functions

length- Return the number of items currently held within our array.

pop()- Remove the last element in our array.

shift()- Remove the first element in our array.

push()- Adds the element to the back of our array.

unshift()- Adds the element to the beginning of our array.

concat()- Merges one array with another.

splice()- Takes 3 arguments, the first one refers to which index we want to start at. The second refers to how many items we will remove. Third argument is the replacement element.

sort()- Sorts the array in alphabetical order.

includes()- Returns true or false if an item is found in the array.
