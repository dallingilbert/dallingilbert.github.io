Further Functions

Call & Apply Methods

  • The call method can be used to set the value of this inside a function to an object that is provided as the first argument
  • The apply method works in the same way, except the arguments of the function are provided as an array
  • The main purpose of these functions is they allow generalized functions to be written that are not tied to specific objects by being methods of that object. This gives flexibility over how the functions can be used.

Memoization

  • If we have a function that takes a lot of computational power, we can store the result in a cache and access it through .cache member function

Recursive functions

  • A recursive function is one that invokes itself until a certain condition is met. It’s a useful tool to use when iterative processes are involved

Promises

  • A promise represents the future result of an asynchronous operation. Promises don't do anything that can't already be achieved using callbacks, but they help simplify the process, and avoid the convoluted code that can result from using multiple callbacks.
  • Promises go through a life cycle, which either ends in being resolved or rejected
  • Once a promise has been settled, it moves on to the then() method to deal with the outcome. Any data returned from the resolve or reject fuction will be passed here.

AJAX

Basic Usage

  • The Fetch API provides a global fetch() method that only has one mandatory argument, which is the URL of the resource you wish to fetch.

Response Interface

  • Deals with the object that’s returned when the promise is fulfilled. Response objects have a number of properties and methods that allow us to process the response effectively.
  • ok - this property checks to see if the response is successful, it does so by checking the HTTP status code
  • headers - A Headers object containing any headers associated with the response
  • url - a string containing the url of the response
  • redirected - a boolean value that specifies if the response is the result of a redirect
