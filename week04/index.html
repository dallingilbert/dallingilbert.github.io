Forms

We can access forms through the legacy DOM, using this method we simply state the syntax as document.forms with an index

Another way to access a form is by the name attribute, this requires that we add a name attribute to our form
Syntax: document.forms['search']

Forms Properties & Methods

form.submit() This method will submit the form automatically

form.action() is used to set the action attribute of a form

Form Events

focus

This event will occur when an element specified is currently being hovered by the users mouse

blur

Occurs when the user moves the focus away from the form element

change

This event triggers when the user moves the focus away from the element after changing it

Retrieving and Changing Values From a Form

Text input element objects have a value property that can be used to retrieve the text inside the field

Input and Placeholder Attribute

We can use the placeholder attribute to specify a default value within a text field if the user has not input text into the field
