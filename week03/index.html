    Objects

    Objects are always created using the curly brackets, and conventionally follow the key: value format.

    Object Functions

    You can extract key value pairs from objects with the Object.entries() function. To access the key and value, we need to use the backtick with ${} format.

    We can also delete properties from an object with the delete key word. This will delete any property of our object using the dot notation

    Copied by Reference

    Objects are copied by reference, therefore if you change one instance of an object, the original copy will be changed as well!

    The Document Object Model

    We can access HTML elements by referencing different document functions such as:

    • document.getElementById
    • document.getElementbyClassName
    • document.querySelector
    • document.querySelectorAll

    querySelectorAll will allows us to specify the index of which element we want to grab, while querySelector can access the first element by the id or name we pass into the function

    Events

    Events allow us to connect user input to the web page. If we have a button, we need to tell the website what to do if that button gets clicked. This is where eventhandlers come into play
