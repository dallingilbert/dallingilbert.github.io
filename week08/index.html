Notes

    CSS Transforms

    • The transform property lets you translate, rotate, scale and/or skew any element on the page.
    • The value of the transform property is one or more transform functions that will be applied in the order they are provided.

    CSS Translations

    • Translation functions allow you to move elements left, right, up or down. When we translate elements, we are moving elements without impacting the flow of the document
    • Example: transform: translate(45px, 45px) (moves the element x pixels from left, and y pixels from the top)
    • Transforms do not work on inline elements. We need the add the display attribute to our CSS with inline-block.

    Scaling

    • The scale function scales an element by the defined factors horizontally then vertically.
    • Sometimes scaling may provide unwanted results, instead of using scaling to properly fit our text, try modifying the element's height, width or font-size instead.

    Rotation

    • The rotate function rotates an element around a point of origin by a specified angle value.

    Transitions

    • Transitions allow us to create a smoothing affect on different animations. Such as having a links color slowly fade to the original after a user hovers over it.
    • It is important to start with a default transition-property. Any property that has a midpoint such as color or width, is able to perform a transition. When we transition we must define a pre-state and a post-state.
